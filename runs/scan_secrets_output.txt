-│ NIVEAU 2: EXTRACTION & MATCHING (AI Intelligence)   │
-├─────────────────────────────────────────────────────┤
-│  • LLM Model 1 : Extrait keywords (skills, domain)  │
-│  • LLM Model 2 : Match offer vs expériences/projects│
-│  • Génère confidence scores (0.0 → 1.0)            │
-│  • Gated Decision : skip/review/go                  │
-└─────────────────────────────────────────────────────┘
-                        ↓
-┌─────────────────────────────────────────────────────┐
-│ NIVEAU 3: GÉNÉRATION MULTI-CANAUX (Output Rendering)│
-├─────────────────────────────────────────────────────┤
-│  • CV personnalisé + Lettre (LaTeX → PDF)           │
-│  • Lettres post-entretien                           │
-│  • Emails de prospection                            │
-│  • Rapports de projets                              │
-│  • Préparation mémoire                              │
-│  • Archive + metadata                               │
-└─────────────────────────────────────────────────────┘
-```

+

+      env:
+        RENDER_API_KEY: ${{ secrets.RENDER_API_KEY }}
+        RENDER_SERVICE_ID: ${{ secrets.RENDER_SERVICE_ID_API }}
+      run: |
+        # Download database backup from Render's storage
+        mkdir -p backups
+        curl -H "Authorization: Bearer $RENDER_API_KEY" \
+          https://api.render.com/v1/services/$RENDER_SERVICE_ID/files/db/recruitment_assistant.db \
+          -o backups/recruitment_assistant_$(date +%Y%m%d_%H%M%S).db
+
+    - name: Compress backup
+      run: |
+        tar -czf backups.tar.gz backups/
+        ls -lh backups.tar.gz
+
+    - name: Upload to GitHub Releases
+      uses: actions/upload-artifact@v4
+      with:
+        name: database-backup
+        path: backups.tar.gz
+        retention-days: 30
+
+    - name: Cleanup old backups
+      run: |
+        # Keep only last 7 daily backups locally
+        ls -t backups/ | tail -n +8 | xargs -r rm

+# API Specification - FastAPI
+
+## Vue d'ensemble
+
+L'API expose les fonctionnalités d'orchestration en 3 niveaux via endpoints REST JSON.
+
+**Base URL:**
+- Local: `http://localhost:8000`
+- Production (Render): `https://recruitment-app.onrender.com`
+
+**Authentification:** À définir (token JWT ou API key)
+
+**Format réponse:** JSON + HTTP status codes standard
+
+---
+
+## Endpoints
+
+### 1. Health & Status
+
+#### `GET /api/health`
+
+Vérification santé de l'API.
+
+**Response (200):**
+```json
+{
+  "status": "healthy",
+  "timestamp": "2026-04-05T10:30:00Z",
+  "db_connected": true,
+  "llm_available": true
+}
+```
+
+---

+### 2. Offers Management
+
+#### `POST /api/offers`
+
+Ingest une nouvelle offre (Markdown).
+
+**Request:**
+```json

+  "source_file": "offer.md",
+  "ingestion_date": "2026-04-05"
+}
+```
+
+**Response (201):**
+```json
+{
+  "status": "ingested",
+  "offer_id": "offer-20260405-xyz123",
+  "company_name": "Acme Corp",
+  "tier": "tier-2",
+  "country": "France",
+  "sections_detected": {
+    "title": "Senior Backend Engineer",
+    "company": "Acme Corp",
+    "responsibilities": ["...", "..."],
+    "skills": ["Python", "FastAPI", "..."]
+  }
+}
+```
+
+**Error (400):**
+```json
+{
+  "error": "Invalid Markdown format",
+  "details": "Missing required section: Tier"

+```

+
+Récupérer offre depuis DB (avec keywords + matching results).

+        with:
+          files: ./coverage.xml

+  build:
+    runs-on: ubuntu-latest
+    needs: test
+

+- Verify RENDER_API_KEY is correct in Render dashboard
+- Check it hasn't expired
+- Regenerate if needed
+
+### GitHub Actions Won't Trigger
+
+**Solution:**
+- Verify `deploy-render.yml` is in `.github/workflows/`
+- Check branch protection rules
+- Manually trigger via "Run workflow" button
+
+### Render Service Not Updating
+
+**Solution:**
+- Check Render dashboard for deployment status
+- View build logs: Render → Service → Deployments
+- Verify environment variables are set in Render UI
+
+### LLM API Key Errors in Logs
+
+**Solution:**
+- Verify OPENAI_API_KEY and CLAUDE_API_KEY are set
+- Confirm they're not expired
+- Check rate limits on API provider dashboards
